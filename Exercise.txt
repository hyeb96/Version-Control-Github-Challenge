1. Explain what the terminal is in your own words.

Terminal is a command line system that allows you to do run commands and access folders and files as well as run commands like open, install, and remove applications. 

2. What is the major difference between Git and Github?

Git is command line tool that is installed locally on your computer while Github stores git repository and allow others to make changes to your code. 

3. Why do we use Github?

We use Github to collaborate and share projects as well as updating our changes to it.
